Proyect_GymFitness es un sistema de base de datos diseñado para gestionar de forma eficiente todas las operaciones internas de un gimnasio.
Incluye control de socios, membresías, entrenadores, clases y horarios, además de reservas, pagos y registro de asistencia.

Tecnologías Utilizadas:
SQL Server (Diseño e implementación de base de datos)
Power BI (Dashboards, KPIs y análisis avanzado)
T-SQL (Consultas, procedimientos y validaciones)
